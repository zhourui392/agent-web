快速开始
  1. （可选）点击顶部工作目录，选择项目根路径
  2. （可选）点击 🌿 分支标签 切换或更新 worktree
  3. 在底部输入框提问，Enter 发送，Ctrl+Enter 换行
输入技巧
  • 输入 / 触发命令补全，↑↓ 选择，Tab 确认
  • 图片：点击「图片」按钮上传，或直接 Ctrl+V 粘贴截图，路径自动追加进消息
  • 附件：点击「附件」按钮上传文本类文件，以 [附件清单] 形式追加进消息供 AI 用 Read 工具读取分析
  • 清除上下文：开启新一轮对话但保留会话
  • 停止：中断当前 AI 回答
侧边栏 / 历史
  • 历史列表展示所有用户的对话，可查看 / 继续他人的会话
  • 点击历史项 查看；点击 ▶ 继续对话；🗑 删除（仅能删除自己创建的对话，他人会话不显示删除按钮）
分享 / 协作
  • 会话头条或历史详情抽屉的 分享 按钮可生成只读公开链接，他人免登录即可查看，但不能续聊或启动 Agent
其他
  • 工作空间弹窗支持 上传 / 下载 / 删除 文件
